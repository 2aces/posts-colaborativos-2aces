A versão mais atual usa shortcodes para criar os links com as tags, as versões futuras permitirão a inserção usando os botões do editor de texto do Wordpress (TinyMCE) Se este plugi te ajuar, consider tuitar sobre a @2AcesConteudo ou dar uma boa pontuação no repositório do Wordpress quando publicarmos por lá, ou, melhor ainda, linkando nosso site no seu blog: www.2aces.com.br

Installation  - WP Affiliates Tagger plugin by 2Aces

  1. Baixe o arquivo aa-affiliates-tagger.zip file neste link ou clone-o  no GitHub, descompacte e faça o upload da pasta /aa-affiliates-tagger/ para o diretório /wp-content/plugins/ de seu blog.
  2. Ative o plugin através do menu Plugins no Wordpress.
  3. Copie as IDs dos programas de afiliados nos campos apropriados em configurações
  4. Se você não ainda não participa de nenhum programa de afiliados, clique no links no painel de configurações para acessar o site dos programas e se cadastrar.
  5. Quando criar o link para uma loja ou serviço, use um shortcode semelhante a este: [[product merchant="merchantname" productlink="productlink"]texto do link[/product]]
  6. Você deve usar minúsculas e não pode usar espaços, acentos ou caracteres especiais nos parâmetros: (e.g. [[product merchant="amazon" productlink="http://www.amazon.com/Web-Marketing-All---One-Dummies/dp/1118243773/"]Web Marketing na Amazon[/product]] )
  7. Para as lojas e serviços Amazon, Americanas, Apple, Cultura, Shoptime, SouBarato, Submarino e Submarino Viagens, você deve o link da página de produto ou no parâmetro productlink: (e.g. [[product merchant="amazon" productlink="http://www.amazon.com/Web-Marketing-All---One-Dummies/dp/1118243773/"]Web Marketing na Amazon[/product]] )
  8. Se você estiver linkando para serviços de hospedagem e gerenciamento como Dreamhost, Media Temple, Manage WP e WP Engine, você não precisa do link de produto e o parâmetro productlink será ignorado. (e.g. [[product merchant="wpengine"]WP Engine evita dores de cabeça![/product]] )

Perguntas Frequentes - Notas

A versão mais atual usa shortcodes para criar os links com as tags. Basicamente, significa que você precisa colocar o texto que você quer com link em pseudo-tags usando chaves [], como neste exemplo: [[product merchant="wpengine"]WP Engine evita dores de cabeça![/product]] Note que dentro da tag de abertura há uma parâmetro indicando qual loja ou serviço está linkando. Alguns deles demanda também o parâmetro productlink.
