--- post_title: 'Amazon’s Publisher Studio Loader para WordPress' author: sadmin2a post_excerpt: "" layout: page permalink: > https://www.2aces.com.br/plugins-wordpress-da-2aces/aa-amazon-studio-loader-portugues/ dsq_thread_id: - "2348735797" --- (english version is found here) Este é um plugin gratuito feito pela 2Aces para tornar mais fácil usar Amazon Publisher Studio num blog Wordpress. O Amazon Publisher Studio é uma nova ferramenta da Amazon que permite que os participantes de seu programa de afiliados criem facilmente links para produtos Amazon diretamente das páginas de seus blogs e sites, eliminando a necessidade de entrar na loja para procurar produtos e criar os links manualmente.

Instalação do plugin Amazon's Publisher Studio Loader

  1. Baixe o arquivo zip do Amazon's Publisher Studio Loader deste link ou de nosso repositório no Github
  2. Unzip aa-amazon-studio-loader.zip and upload aa-amazon-studio-loader folder to the /wp-content/plugins/ directory
  3. Ative o plugin através do menu plugins no Wordpress
  4. Vá até seu perfil no Amazon Associates e selecione a aba Publisher Studio ( cliqie aqui )
  5. Crie uma nova instância (new instance) e clique em "Get Code"
  6. Procure no código pela sua Publisher tag. É um código alftanumérico entre aspas depois do texto codeamznpubstudioTag. ( i.e cybl0a-20 em window.amznpubstudioTag = "cybl0a-20"). Copie este código e cole no primeiro campo da página de configuração do plugin.
  7. Procure no código pelo nome de arquivo de seu script customizado. É um código alfanumérico entre "http://ps-us.amazon-adsystem.com/domains/" e ".js" ( e.g.cybl0a-20_a5e1c142-fc0b-4b4c-abf8-27814868e9b8 , em http://ps-us.amazon-adsystem.com/domains/cybl0a-20_a5e1c142-fc0b-4b4c-abf8-27814868e9b8.js ) Copie este código e cole no segundo campo da página de configuração do plugin.
