Amazon Publisher Studio it's a new tool from Amazon that allow Amazon Associates affiliates to easily create links to Amazon products directly from their blog/sites front-end. Eliminating the need of going to Amazon to search for products and manually placing the links.

Installing Amazon's Publisher Studio Loader Plugin

  1. Download Amazon's Publisher Studio Loader zip file from this link, or from our Github repository
  2. Unzip aa-amazon-studio-loader.zip and upload aa-amazon-studio-loader folder to the /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress
  4. Go to your Amazon Associates profile and select the Publisher Studio tab ( click here )
  5. Create a new instance and click on "Get Code"
  6. Search the code and copy your Publisher tag. This is the alphanumeric code between the double quotes after the codeamznpubstudioTag. ( i.e cybl0a-20 on window.amznpubstudioTag = "cybl0a-20") and paste in the first field of the plugin's settings page
  7. Search the code and copy your custom script file name. This is the alphanumeric code between http://ps-us.amazon-adsystem.com/domains/ and .js. ( e.g.cybl0a-20_a5e1c142-fc0b-4b4c-abf8-27814868e9b8 , on http://ps-us.amazon-adsystem.com/domains/cybl0a-20_a5e1c142-fc0b-4b4c-abf8-27814868e9b8.js ) and paste in the second field of the plugin's settings page
