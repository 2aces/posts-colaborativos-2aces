Woocommerce Radio Buttons - International Version

(Versão em português acima) One of best things about Wordpress it's the use of plugins to make it better or to fit your neeed. Another one is the extense developers and contributors community creating and improving the work of each other. Well, today we contribute a little bit to this ecosystem, customizing a plugin which has been proving itself useful to us: Woocommerce Radio Buttons, by DesignLoud. A plugin which makes possible for use radio buttons, instead of the default drop-down menu, to select attributes on Woocommerce product pages (Woocommerce is an e-commerce plugin) Unfortunately, so far, the original plugin is not internationalization friendly and the options/attributes header appears only in english (Choose an option...) Infelizmente, o plugin original não é amigável à localização e o cabeçalho das opções só aparecia em inglês. In our version, there are 2 changes: [caption id="attachment_547" align="aligncenter" width="640"]Woocommerce Radio Buttons - International Version, on Fotospot Gallery website[/caption]
  • You can translate the option header
  • The option series names is also printed in the screen (e.g. Size, Color, etc)
You can see the plugin in action in 2 clients projects: Fotospot - Fotografia Contemporânea (fine art photography gallery) and Fotografias Aéreas (aerial photography image bank). While the original plugin author doesn't merge these changes to the official version, you can download our customized version clicking here.
